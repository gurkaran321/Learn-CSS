Terminology

  • Content Box: This is the area where the content is displayed on the webpage
  • Padding Box: Padding is inside the box and it is the space between content and border of the box added by using padding property.
  • Border Box: Border surrounds the box and visually gives a frame to the box
  • Margin Box: Margin is the space around the box and this doesnot add to the size of the box
  • Debugging: This is the most important usage of box model along with DevTools.
  • Padding, border and margin each have their dedicated CSS property which is used to style them.
  • You can add box-model to inline elements by using display:block CSS property.
  • There is a CSS property which controls sizing of box called box-sizing.
